The purpose of this website is to immitate basic functionalities of a common posting system. The main site introduces the developer and contains three buttons that offer a simple navigation system between the main page, a posting system page and a post search page.

Q1. Special Features:
Expression of an html website inside a modal, this was accomplished with JS.
Styling, even though Bootstrap provides considerable amounts of styling, I found it necessary to jump onto bootstrap.css and edit to my likes which I believed improved the look of the website.

Q2. Parts I had Trouble:
Attempting to display an entire website on a div without including libraries seemed a bit of a challenge. At first I attempted to create an object and pass it an html, then load it as soon as the modal was opened. Unfortunately my function was not displaying the object properly and styling had no positive effects. At the end I decided to go to W3Schools to create a function that worked.

Q3. What would I do Better:
Im unexperienced with back end but I know how Globals can be abused. I'd like to know how to implement security.

Q4. My Sources:
